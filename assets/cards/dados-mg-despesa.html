Despesa pública do poder executivo do Estado de Minas Gerais

Resources

Ação

Tabela que agrega dados das ações, que são conjuntos de operações das quais resulta a entrega de um produto (bem, serviço ou transferência financeira) que contribui para atender ao objetivo de um programa.

Categoria Econômica

Tabela que agrega dados da Classificação Econômica da despesa: corrente (funcionamento dos órgãos, pagamentos dos servidores, etc) ou de capital (obras, equipamentos, etc)

Elemento de despesa

Tabela que agrega dados do Elemento de Despesa, que é a classificação da despesa quanto ao objeto do gasto (diárias, material de consumo, pensões, locação de imóveis, etc.).

Empenho

Tabela que contém dados sobre os empenhos

Fonte

Tabela com dados da classificação da origem dos recursos utilizados para realizar esta despesa.

Função

Tabela com dados da classificação das áreas de atuação do setor público nas quais são gastos os recursos, tais como saúde, educação, administração, transportes, etc.

Grupo da despesa

Tabela que contém dados da classificação da despesa quanto ao objeto do gasto (pessoal, juros, amortização da dívida, investimentos, etc.).

Item de despesa

Tabela que contém dados sobre o item de despesa, que é a classificação da despesa quanto ao desdobramento do elemento de despesa (material gráfico, medicamentos, diárias, etc).

Modalidade de aplicação

Tabela com dados da modalidade de aplicação, que é a classificação que indica se os recursos são aplicados diretamente por órgãos ou entidades no âmbito da mesma esfera de Governo ou por outro ente da Federação.

Procedência

Tabela com dados da procedência dos recursos orçamentários. Ex.: se os recursos são para livre utilização, se foram transferidos de outra unidade orçamentária, se são para contrapartida a convênios, etc.

Programa

Tabela com dados dos programas, que são conjuntos de ações para atender um problema ou demanda da sociedade.

Subfunção

Tabela com dados da subfunção, que identifica a natureza básica das ações que se aglutinam em funções.

Tempo Diário

Tabela com dados de data

Tipo de documento

No description provided

Unidade orçamentária

Tabela com dados sobre a unidade orçamentária responsável pela despesa

Favorecido

Tabela com dados do beneficiário da despesa
